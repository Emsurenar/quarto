# Quarto Ljudfiler

Lägg dina anpassade ljudfiler här! 

Filerna som förväntas är:
- `woah.mp3` – Spelas upp vid ett "Woah!"-kudos.
- `namen.mp3` – Spelas upp vid ett "Nämen!"-kudos.
- `omojligt.mp3` – Spelas upp vid "Det är inte möjligt!"-kudos och när spelare gör ett ogiltigt drag (error toasts).

Om filerna saknas eller inte kan spelas upp, kommer spelet automatiskt att falla tillbaka på de inbyggda syntetiserade ljuden eller vara tyst.
